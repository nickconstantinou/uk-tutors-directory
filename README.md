# UK Tutors Directory

A UK-based tutor directory for connecting parents with tutors. Built with the Greg Isenberg directory methodology.

## Quick Start

1. Clone this repo
2. Open `index.html` in your browser
3. That's it - no server needed!

## Features

- **Search** by subject, location, and level
- **Filter** tutors by GCSE, A-Level, Primary, etc.
- **Tutor profiles** with ratings and prices
- **Listing form** for tutors to sign up

## Tech Stack

- Pure HTML/CSS/JS (no framework)
- GitHub Pages for hosting
- localStorage for data (MVP)

## Monetisation

- Lead generation: £30-50 per lead
- Premium listings: £20-50/month
- Cross-sell: ExamPulse integration
